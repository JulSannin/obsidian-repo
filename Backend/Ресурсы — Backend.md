# 📚 Ресурсы — Backend

[[Backend Roadmap|← карта]] · frontend-аналог: [[Ресурсы — Frontend]]

> [!tip] Как выбирать
> Официальная документация + один основной курс на тему. Backend особенно требует практики — читай меньше, кодь больше.

## 🌐 Общее / маршрут
- **roadmap.sh/backend** и **roadmap.sh/nodejs** — интерактивные карты (англ.).
- **The Odin Project** (путь NodeJS) — бесплатный, проектно-ориентированный.
- **MDN: Express/Node tutorial** — хороший вводный курс.

## 📖 По темам

### Node.js / Express → [[Node.js]]
- **nodejs.org/docs**, **expressjs.com** — официальные доки.
- Курсы: The Odin Project, freeCodeCamp (Node на YouTube).

### Базы данных / SQL → [[Базы данных и SQL]]
- **SQLBolt**, **PostgreSQL Tutorial** (postgresqltutorial.com) — интерактив.
- **Prisma docs** — отличная документация с примерами.
- Книга: «SQL за 10 минут» (Бен Форта).

### REST API → [[REST API на сервере]]
- «REST API Design Best Practices», **Swagger/OpenAPI** docs, **Zod** docs.

### Аутентификация / безопасность → [[Аутентификация и безопасность]]
- **OWASP Top-10** (owasp.org) — обязательно прочитать.
- **jwt.io** — разобрать JWT руками.
- Доки **bcrypt**, **Passport.js**, **helmet**.

### Архитектура / DevOps → [[Серверная архитектура и DevOps]]
- **Docker docs** + «Docker для начинающих» (YouTube).
- **The Twelve-Factor App** (12factor.net) — принципы прод-приложений.
- **GitHub Actions docs**, доки Railway / Render / Fly.io.

## 🇷🇺 Русскоязычное
- **learn.javascript.ru** — раздел про Node частично, плюс вся база JS.
- Курсы: Хекслет (backend на Node), Яндекс Практикум (веб-разработчик).
- YouTube: Ulbi TV, Владилен Минин (Node/Express), freeCodeCamp (рус. субтитры).

## 🏋️ Практика
- **Codewars / LeetCode** — алгоритмы (общие с [[Алгоритмы и структуры данных]]).
- Строй API из [[Backend-проекты]] — это и есть главная практика.

> [!warning] Не застревай в теории
> Backend нельзя выучить чтением. Подними сервер на первой неделе и наращивай его по мере прохождения тем.
