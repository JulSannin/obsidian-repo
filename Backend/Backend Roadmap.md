# ⚙️ Backend Roadmap: Intern → Junior

[[Карта обучения|🏠 общий хаб]] · смежный трек: [[Frontend Roadmap]]

> [!abstract] О чём эта карта
> Маршрут backend-разработчика от стажёра до джуниора. Стек — **Node.js + Express + PostgreSQL** как лучший мост для фронтендера (тот же язык — JavaScript/TypeScript). **Фундамент универсален** (БД, API, авторизация, архитектура), поэтому стек при желании заменяется: Python/Django, Java/Spring, Go, C#/.NET, PHP/Laravel.

> [!tip] Предусловие
> Заходи сюда, когда уверен в [[JavaScript]] (хотя бы уровень [[Frontend · Junior]]). Backend на Node — это тот же JS, только на сервере.

## 🎯 Что отличает уровни
> [!info] Intern → Junior
> - **Intern** — поднимает сервер по инструкции, делает простые запросы к БД, понимает, как клиент общается с сервером.
> - **Junior** — самостоятельно пишет REST API с базой и авторизацией, обрабатывает ошибки и валидирует ввод, деплоит проект.

> [!warning] Антипаттерны
> - Учить backend без уверенного JS → каша в голове. Сначала [[JavaScript]].
> - Игнорировать **базы данных** — это сердце backend, важнее фреймворка.
> - Хранить пароли в открытом виде, секреты в коде → см. [[Аутентификация и безопасность]].
> - Не обрабатывать ошибки и не валидировать вход — главный признак джуна-новичка.
