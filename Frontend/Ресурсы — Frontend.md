# 📚 Ресурсы — Frontend

[[Frontend Roadmap|← карта]] · backend-аналог: [[Ресурсы — Backend]]

> [!tip] Как выбирать
> Не распыляйся. Один основной источник на тему + официальная документация. Документация почти всегда лучше курсов — учись её читать.

## 🌐 Главные справочники
- **MDN Web Docs** — справочник №1 по HTML/CSS/JS. → [[HTML и доступность]] · [[CSS и вёрстка]] · [[JavaScript]]
- **web.dev** (Google) — современные практики, производительность, a11y.
- **roadmap.sh/frontend** — интерактивная карта (англ.), полезна как чеклист.
- **caniuse.com** — поддержка фич браузерами.

## 📖 По темам

### JavaScript → [[JavaScript]]
- **learn.javascript.ru** (Илья Кантор) — лучший русскоязычный учебник.
- **javascript.info** — он же на английском.
- Книги: «You Don't Know JS» (Kyle Simpson), «Eloquent JavaScript».

### CSS → [[CSS и вёрстка]]
- **Kevin Powell** (YouTube) — лучший контент по CSS.
- **css-tricks.com**, игры: Flexbox Froggy, Grid Garden.

### TypeScript → [[TypeScript]]
- **TypeScript Handbook** (офиц.).
- **Total TypeScript** (Matt Pocock), **type-challenges** (GitHub).

### React → [[React и экосистема]]
- **react.dev** — официальные доки (новые, отличные, с интерактивом).
- Доки **TanStack Query**, **Zustand**, **Redux Toolkit**.
- **Next.js Learn** — официальный курс.

### Git → [[Git и инструменты]]
- **Pro Git** (бесплатная книга) · **learngitbranching.js.org** (интерактив).

### Сеть → [[Сеть и HTTP]]
- **MDN: HTTP**, «High Performance Browser Networking» (бесплатно онлайн).

### Тестирование → [[Тестирование]]
- Доки **Testing Library**, **Vitest**, **Playwright**; Kent C. Dodds.

### Архитектура → [[Архитектура и паттерны]]
- **feature-sliced.design**, **refactoring.guru**, «Чистый код» (Р. Мартин).

## 🇷🇺 Русскоязычные платформы и сообщества
- Курсы: HTML Academy, Хекслет, Яндекс Практикум, Школа 21.
- Telegram-каналы и чаты по фронтенду, локальные митапы.
- YouTube: Ulbi TV, Владилен Минин, Михаил Непомнящий и др.

## 💪 Практика и алгоритмы
- **Frontend Mentor** — реальные макеты для вёрстки.
- **Codewars / LeetCode (easy)** — алгоритмы для собеса.
- **JavaScript30** — 30 проектов на чистом JS.

> [!warning] Tutorial hell
> Курсы — это вход, не цель. Как только понял основы темы — закрывай курс и строй своё → [[Frontend-проекты]]. Знание появляется в практике, а не в просмотре.
