# 🛠️ Frontend-проекты

[[Frontend Roadmap|← карта]]

> [!tip] Главное правило
> **Учишься — строишь.** Проекты в портфолио важнее сертификатов. 3–4 завершённых и задеплоенных проекта убедительнее, чем 20 туториалов. Каждый проект → README + живой деплой + ссылка на GitHub.

## 🟢 Intern → [[Frontend · Intern]]
Цель: чистая вёрстка по макету.
- **Лендинг по макету из Figma** — pixel perfect, адаптив.
- **Личная страница-визитка** — деплой на GitHub Pages.
- **Промо-страница продукта** с формой и анимациями на CSS.
- Тренажёры: Flexbox Froggy, Grid Garden, CSS Battle.

## 🟡 Junior → [[Frontend · Junior]]
Цель: интерактив на React + данные из API.
- **Todo-приложение** с фильтрами и сохранением в localStorage.
- **Погода/Фильмы/Рецепты** — потребление публичного REST API (OpenWeather, TMDB).
- **Калькулятор / конвертер валют**.
- **Лента постов** с пагинацией и поиском.
- Деплой на **Vercel / Netlify**.

## 🟠 Junior+ → [[Frontend · Junior+]]
Цель: полноценное приложение на React + TS + тесты.
- **Интернет-магазин (mini)**: каталог, корзина, фильтры, состояние (Zustand/RTK).
- **Канбан-доска** с drag-n-drop.
- **Дашборд с авторизацией**: защищённые роуты, JWT, CRUD.
- **Клиент к своему/публичному API** с TanStack Query.
- Обязательно: TypeScript + unit-тесты → [[Тестирование]], структура по [[Архитектура и паттерны]].

## 🔵 Middle → [[Frontend · Middle]]
Цель: уровень продакшена.
- **Полноценный продукт** на Next.js: SSR, авторизация, оплата (тест), админка.
- **Реал-тайм приложение**: чат на WebSockets.
- **Вклад в Open Source** — реальные PR в чужой кодовой базе (это сильнейший сигнал для работодателя).
- Настрой **CI/CD**, мониторинг (Sentry), e2e-тесты.
- 🟣 **Fullstack-проект**: подключи свой React к собственному backend → [[Backend-проекты]].

## 📋 Чек-лист хорошего проекта
- [ ] Внятный README: что это, как запустить, стек, скриншоты.
- [ ] Живой деплой (ссылка работает).
- [ ] Чистая история коммитов.
- [ ] Адаптив и базовая доступность.
- [ ] Обработаны загрузка, ошибки, пустые состояния.
- [ ] (с Junior+) Типы и тесты.

> [!quote] Идея для самого ценного проекта
> Реши **свою реальную проблему**. Такой проект ты доведёшь до конца и сможешь живо рассказать о нём на собеседовании — это всегда заметно.
