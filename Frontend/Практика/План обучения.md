# 📅 План обучения

[[Frontend Roadmap|← карта]] · прогресс отмечай в [[Frontend — чеклист навыков]]

> [!tip] Формула
> **70% практика / 30% теория.** Регулярность важнее объёма: лучше 1–1.5 часа каждый день, чем 10 часов в воскресенье. Учишь тему → сразу применяешь в [[Frontend-проекты]].

## 🗓️ Структура недели (пример, ~12–15 ч)
| День | Фокус |
|------|-------|
| Пн | Новая тема: теория + конспект (этот vault) |
| Вт | Практика по теме (мини-задачи) |
| Ср | Работа над текущим проектом |
| Чт | Новая тема / углубление |
| Пт | Проект + рефакторинг |
| Сб | Алгоритмы ([[Алгоритмы и структуры данных]]) + повторение |
| Вс | Отдых или лёгкое чтение / разбор чужого кода |

## ⏱️ Дневная сессия (Pomodoro)
- 4 × 25 минут с перерывами 5 мин.
- 1 блок — теория, 3 блока — код.
- В конце: 1 предложение в дневник «что понял сегодня».

## 📈 Привычки, которые ускоряют
- **Учись на людях** (build in public): выкладывай прогресс — мотивирует и строит сеть контактов.
- **Интервальное повторение**: возвращайся к старым темам через неделю/месяц.
- **Объясняй вслух** (резиновая уточка / пост): не понял — не сможешь объяснить.
- **Ревью своего кода** через неделю: увидишь, что улучшить.

## 🚧 Главные ловушки
- [!] **Tutorial hell** — бесконечные курсы без своего кода. Строй с первой недели.
- [!] Прыжки между технологиями без завершения. Доводи проект до конца.
- [!] Сравнение с другими. Сравнивай себя с собой вчерашним.

> [!note] Метрика прогресса
> Не «сколько часов просмотрел», а **«что я теперь умею делать сам»**. Сверяйся с [[Frontend — чеклист навыков]] раз в 2 недели.
